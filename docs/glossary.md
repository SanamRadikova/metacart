# MetaCart — Glossary

This document defines key terms used throughout the project.

## 🧬 Biomarkers

| Term | Definition | Standard Unit |
|------|------------|---------------|
| **Glucose (fasting)** | Blood sugar level after 8+ hours of fasting | mg/dL |
| **HbA1c** | Average blood sugar over 3 months | % (NGSP) or mmol/mol (IFCC) |
| **Triglycerides (TG)** | Fat in the blood | mg/dL |
| **HDL** | "Good" cholesterol | mg/dL |
| **LDL** | "Bad" cholesterol | mg/dL |
| **hs-CRP** | High-sensitivity C-reactive protein (inflammation marker) | mg/L |
| **TSH** | Thyroid-stimulating hormone | mIU/L |
| **HRV RMSSD** | Heart rate variability — root mean square of successive differences | ms |
| **SDNN** | Standard deviation of all NN intervals | ms |
| **PNN50** | Percentage of successive NN intervals that differ by more than 50 ms | % |
| **dG/dt** | Rate of change of glucose (glucose velocity) | mg/dL/min |

## 🎯 Axes

| Axis | Name | What It Measures |
|------|------|------------------|
| **Axis 1** | Glycemic | Carbohydrate tolerance (glucose, HbA1c, TG/HDL ratio) |
| **Axis 2** | Lipid | Atherogenic lipid pattern (TG, HDL) |
| **Axis 3** | Inflammatory | Systemic chronic inflammation (hs-CRP) |
| **Axis 4** | Stress / Thyroid | Thyroid function and stress adaptation (TSH) |
| **Axis 5** | Neuro-Autonomic | Nervous system balance (HRV RMSSD, SDNN, PNN50) |

## 🧬 Profiles

| Profile | Name | When Activated |
|---------|------|----------------|
| **Profile 1** | Metabolic Flexibility | All 5 axes stable, HRV normal |
| **Profile 2** | Carb Sensitivity | Axis 1 or 2 elevated, HRV normal |
| **Profile 3** | Inflammatory Load | Axis 3 elevated, axes 4-5 not elevated |
| **Profile 4** | Stress-Adaptive | Axis 4 elevated, Profile 5 not activated |
| **Profile 5** | Neuro-Autonomic | Axis 5 elevated OR symptoms on dG/dt OR hormonal modifier active |

## 📊 Key Concepts

| Term | Definition |
|------|------------|
| **Drift** | Difference between recommended cart and actual purchases |
| **Grocery Stability Score** | Proprietary score measuring consistency of grocery purchases over time |
| **Graceful Degradation** | Engine continues to work with missing data (marks axis as `no_data`) |
| **Functional Thresholds** | Stricter than diagnostic criteria, used for preventive nutrition |
| **Cultural Food Patterns** | Traditional foods specific to a cultural group (e.g., borscht for Eastern European) |
| **Timepoint** | A specific measurement occasion (baseline, follow-up) |
| **Cohort** | Group of users in the pilot (standard or deep tracking) |

## 🛒 Cart Terms

| Term | Definition |
|------|------------|
| **Recommended Cart** | Grocery list generated by MetaCart based on profile |
| **Actual Purchase** | What the user actually bought (captured manually or via receipt) |
| **Trade Unit** | How products are sold in stores (e.g., "18 ct eggs", "32 oz yogurt") |
| **Budget Tier** | Pricing level (LOW = Walmart, MID = Costco, HIGH = Whole Foods) |
| **Household Scaling** | Adjusting quantities based on number of people in household |

## 🔬 Research Terms

| Term | Definition |
|------|------------|
| **Deep Tracking Cohort** | ~50-75 users with CGM + HRV (continuous monitoring) |
| **Standard Cohort** | ~200 users with labs + grocery tracking (no CGM) |
| **Baseline** | First measurement (before intervention) |
| **Follow-up** | Second measurement (after intervention) |
| **IRB** | Institutional Review Board (ethics committee) |
| **CGM** | Continuous Glucose Monitor (e.g., Stelo, Dexcom, Libre) |

## 📱 Technical Terms

| Term | Definition |
|------|------------|
| **OpenAPI** | Specification for describing REST APIs (formerly Swagger) |
| **Hypertable** | TimescaleDB table optimized for time-series data |
| **DTO** | Data Transfer Object (used for API communication) |
| **BLoC** | Business Logic Component (Flutter state management pattern) |
| **Modular Monolith** | Monolithic application with clear module boundaries |

## 🌍 Cultural Groups

| Group | DB Value (snake_case) | Description |
|-------|----------------------|-------------|
| **Eastern European** | `eastern_european` | Borscht, bone broth soups, fermented foods, buckwheat |
| **South Asian** | `south_asian` | Rice & dal, roti, spiced vegetables, chai |
| **Latino** | `latino` | Beans & rice, corn tortillas, fresh salsas |
| **African-American** | `african_american` | Collard greens, sweet potato, black-eyed peas |
| **East Asian / Chinese** | `east_asian` | Rice & greens, tofu, miso, noodles, dim sum |
| **Standard American** | `standard_american` | Store-brand, frozen, processed (fallback) |
